Shopping List App
---------------------------

This is a web application that allows users to create a shopping list for food, clothing, and gadgets. 
The app utilizes local storage to store the items added to each list.

Getting Started
---------------------------

To use the app, simply open the index.html file in a web browser. 
The app will load and present the user with three separate lists for food, clothing, and gadgets.

Usage
---------------------------

To add an item to the list, enter the item name into the corresponding input field and press "Enter" or click the "Add" button. 
The item will be added to the list and saved to local storage.

To mark an item as complete, check the checkbox next to the item's name. The item will be crossed out to indicate that it is complete.

To remove an item from the list, click the trash can icon next to the item's name.

To clear all items from the list and local storage, click the "Clear All" button.

Naming
---------------------------

This app is called the Shopping List App because it allows users to create a shopping list for different categories, including food, clothing, and gadgets. 
The app utilizes local storage to store the items added to each list.
